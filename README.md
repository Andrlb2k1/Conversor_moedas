# Conversor_moedas
Aplicativo para celular que converte moedas, feito com HTML, CSS, JavaScript, React Native, JSON, Node.js e Expo.

Comandos:

npm install -g expo-cli

npx create-expo-app --template blank
- Nome: conversor-moedas

cd conversor-moedas

npm run android

npm run ios

npm run web

npx expo start

npx expo start --tunnel
